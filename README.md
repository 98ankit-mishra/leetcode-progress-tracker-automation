# LeetCode Progress Tracker Automation 🚀

An automated workflow built using n8n that tracks LeetCode progress daily.

## Features
- Fetches LeetCode stats automatically
- Stores data in Google Sheets
- Sends Telegram reminders
- Tracks solved problems daily
- Fully automated workflow

## Tech Stack
- n8n
- Google Sheets API
- Telegram Bot API
- GraphQL API

## Workflow
Scheduler → Fetch LeetCode Stats → Save to Google Sheets → Send Telegram Notification
